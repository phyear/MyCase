# 基于springboot+thmeleaf+bootstrap的权限管理
* 开发语言：java
* 开发工具：STS
* 编码集:utf-8
* 数据库：MySql
* 服务器中间件：tomcat 

## 1.实现功能
   * #### 部门管理
   * #### 角色管理
   * #### 权限管理
   * #### 菜单管理
   * #### 用户管理

> * 下载maven项目
> * 在eclipes或STS中导入
> * 导入resource文件夹下sql文件到数据库
> * 修改数据库配置文件
> * 运行
